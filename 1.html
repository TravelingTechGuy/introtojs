What is JavaScript?

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive (ex. having complex animations, clickable buttons, popup menus, etc.). There are also more advanced server side versions of JavaScript such as Node.Js which allow you to add more functionality to a website...

Mozilla

  • Has nothing to do with the Java language - its naming is a marketing ploy

JavaScript main features

  • High level (abstracts from the details of the computer or environment)
  • Interperted (rather then compiled)
  • Object oriented (more about that in week 3)
  • Multi-platform (can run on any machine, OS, or device - both on the client and server side)
  • Most features of the language work in every browser for exceptions check CanIUse

JavaScript Interpreter

Most applications we use (such as this browser) are compiled, meaning a special program called compiler processed the source code into machine language.
While compiled applications are (genrally speaking) faster, they are tied to a specific platform (OS, or computer), are usually large files, and are hard to debug once compiled.

An interpreter executes each line of code as it reads it. This allows us to use the same source code for many platforms (in our case, browsers), the source code is relatively small (and can be further minimized and compressed), and is much easier to debug in run-time.
Your browser has a JavaScript engine, that acts as an interpreter.

Common JavaScript engines:

  • V8 - Chrome/Chromium-based browsers/Node JS/Android
  • SpiderMonkey - Firefox
  • Chakra - Microsoft Edge
  • JavaScriptCore - Safari/WebKit based browsers/iOS

The REPL

JavaScript is an interpreted programming language, meaning the JavaScript engine in your browser takes one instruction at a time, interprets it into bytecodes, and executes it. That makes the languge dynamic and fast, and allows for all kinds of optimizations.

JavaScript provides a REPL (Read-Eval-Print Loop) that allows us to freely type JavaScript instructions and have them executed right away.

You have a REPL like that in your browser, in your Developer Tools. Let's access it:

  • In Chrome, go to Menu > More Tools > Developer Tools
    or click CTRL+SHIFT+I (CMD+SHIFT+I on Mac)
    or right-click the mouse anywhere on the page, and select Inspect
  • In Firefox go to menu > Web Develop >Toggle Tools
    or click CTRL+SHIFT+I (CMD+SHIFT+I on Mac) or right-click the mouse anywhere on the page, and select Inspect Element
  • In Microsoft Edge select Developer Tools from the menu, or click F12
    or right-click the mouse anywhere on the page, and select Inspect
  • In Safari Choose Safari > Preferences, and click Advanced
    At the bottom of the pane, select the “Show Develop menu in menu bar” checkbox
    Choose Develop > Show Web Inspector

In all browsers, you'll be able to switch to a tab named Console
In Chrome, you may need to switch to the Sources tab, before Console becomes available.

Things you can type into the REPL:

  • Numbers
  • Numeric operations (try typing 1+1 or 2*5)
  • Strings - a set of characters surrounded by quotes or double quotes
  • Console functions - try typing console.log('hello')
  • Functions - try typing Math.sqrt(4)
  • Assignments - try typing a = 'my name'
