Week 2

The console is nice, but if we want to be able to use JavaScript in an HTML page, we need to be able to embed it.

Here's a simple HTML page that says Hello:

<html>
<head>
  <title>I'm a page</title>
</head>
<body>
  Hello
</body>
</html>

Type it into your text editor, save it as test.html, then open it in your browser

Now let's add some JavaScript. On the page, before the line </body> (line 7)
Type the following: <script>document.write(' world!');</script>
Save the file, refresh the browser. What do you see?
